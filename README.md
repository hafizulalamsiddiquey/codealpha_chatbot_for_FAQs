# 🤖 FAQ Chatbot

A simple AI-powered FAQ Chatbot developed as part of the **CodeAlpha AI Internship – Task 02**.

The chatbot answers frequently asked questions related to Artificial Intelligence, Machine Learning, NLP, Deep Learning, Computer Vision, Python, and CodeAlpha using **TF-IDF Vectorization** and **Cosine Similarity**.

---

## 📌 Project Overview

The FAQ Chatbot uses Natural Language Processing (NLP) techniques to understand user questions and return the most relevant answer from a predefined FAQ dataset.

---

## ✨ Features

- 🤖 Interactive chatbot interface
- 💬 AI-related FAQs
- 🔍 NLP preprocessing
- 📊 TF-IDF Vectorization
- 📈 Cosine Similarity matching
- ⚡ Fast response generation
- 📱 Responsive UI
- ⌨️ Enter key support
- 📂 Category-based FAQ navigation

## ⚙️ Working

1. User types a question.
2. Text is preprocessed.
3. TF-IDF vectors are generated.
4. Cosine Similarity compares the question with stored FAQs.
5. The most similar FAQ answer is displayed.
